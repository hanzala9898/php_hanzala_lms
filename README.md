A modern, clean, and fast Student Management System built with **PHP**. This project includes interactive DataTables, and a fully functional CRUD (Create, Read, Update, Delete) system.

This project is developed using PHP and MySQL.
It includes features like:
- Student Management
- Teacher Management
- Course Management
